This page is dedicated to General Help with DLB, for diagnosing problems and a mini-index to the Command Help sections


Commom Questions:
'Where does DLB show user level up's?'
DLB sends messages in a channel called 'Levels' (caution: Capital L). If one doesn't exist, it will create one, but with basic permissions handled.
'How long does it take to cumulate coins and XP?'
Coins and XP are automatically given to users every 60 seconds. The amount of coins given is a random interger between 1-5, while the amount of XP is a random interger between 25-35.
